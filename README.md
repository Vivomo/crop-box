Crop Box
=========================

## Installation

```
npm install @vimo/crop-box
```

## API


## example
```html
<link rel="stylesheet" href="dist/crop-box.css">
<style>
#wrap {
position: relative;
height: 600px;
}
</style>
<div id="wrap"></div>
<script src="dist/crop-box.js"></script>
<script >
new CropBox({
    elem: document.getElementById('wrap')
});
</script>
```
